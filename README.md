# AuthEveryDay 🔐

**AuthEveryDay** — это ультимативный плагин авторизации нового поколения для серверов **Paper / Spigot / Folia / Bungee / Velocity**. Написан полностью на **Kotlin** под **Java 25**, плагин представляет собой отказоустойчивую экосистему мультифакторной защиты (MFA/3FA) корпоративного уровня с глубокой двусторонней синхронизацией между игровым сервером и Discord Gateway API.

Плагин разработан как легковесная, современная и безопасная альтернатива AuthMe с упором на максимальный комфорт игроков и передовые технологии защиты от взломов.

---

## ✨ Ключевые особенности

* 🚀 **Технологический стек будущего:** Скомпилирован под Java 25 и Kotlin для работы на новейших и самых быстрых сборках Purpur/Paper.
* 📦 **Полная автономность (ShadowJar):** Все зависимости (включая JDA и GoogleAuth) вшиты внутрь плагина и изолированы (relocated), что гарантирует отсутствие конфликтов с другими плагинами на сервере.
* 📂 **Оптимизированное хранилище SQLite:** Быстрая локальная база данных `auth.db`. Имена пользователей регистронезависимы для предотвращения любых векторов обхода авторизации.
* 📱 **Архитектура 3FA (Мультифакторная защита):** Полная свобода выбора для игрока. Защита аккаунта может осуществляться как через Discord, так и через приложения аутентификации (**Google / Microsoft Authenticator, Яндекс Ключ** и др.) по криптографическому протоколу TOTP.
* 💬 **Вход в App без команд:** Игроку больше не нужно писать длинные команды в чат. При запросе TOTP-кода достаточно просто написать 6 цифр из приложения в обычный игровой чат. Плагин перехватит, проверит и скроет сообщение от других игроков.
* 🎛️ **Динамические Discord-компоненты (Панель Сессии):**
  1. **При входе:** Бот отправляет сообщение с IP, GeoIP (Страна/Город) и **точными координатами нахождения игрока в мире**, прикрепляя всего 2 кнопки: `✅ Авторизовать` и `🚨 Заблокировать IP` (вечный бан IP в SQLite и моментальный кик).
  2. **Во время игры:** Как только игрок авторизовался (кнопкой или кодом в чате), сообщение в Discord асинхронно редактируется: конфиденциальный лог координат стирается, а интерфейс превращается в панель управления активной сессией с кнопкой **`🚪 Кикнуть с сервера`**.
  3. **При выходе:** При отключении игрока от сервера кнопки удаляются, а сообщение меняется на: *❌ Сессия завершена. Игрок вышел с сервера*.
* 🛑 **Защита на уровне PreLogin:** Заблокированные через Discord злоумышленники отсекаются сервером асинхронно на этапе рукопожатия (`AsyncPlayerPreLoginEvent`), не успевая нагрузить машину и прогрузить мир.
* 🕒 **Умные временные сессии:** Настраиваемое время жизни сессии (например, 5 минут). При крашах или перезаходах в игру в этом промежутке повторный ввод пароля и 2FA не требуются.

## 🛠️ Команды и Права

| Команда | Описание | Права |
| :--- | :--- | :--- |
| `/register <пароль>` | Регистрация нового аккаунта (без привязки соцсетей) | `autheveryday.use` |
| `/login <пароль>` | Авторизация и запуск привязанных этапов проверки (2FA/MFA) | `autheveryday.use` |
| `/2fa_discord <ID>` | Привязать/изменить свой Discord ID для входа по кнопке | `autheveryday.use` |
| `/2fa_discord unlink` | Отвязать текущий аккаунт Discord от профиля | `autheveryday.use` |
| `/2fa_app setup` | Сгенерировать секретный Base32-ключ для Microsoft/Google App | `autheveryday.use` |
| `/2fa_app confirm <код>`| Подтвердить и активировать привязку приложения Authenticator | `autheveryday.use` |
| `/2fa_app unlink` | Полностью отвязать приложение Authenticator от профиля | `autheveryday.use` |
