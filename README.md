# AuthEveryDayG 🔐

**AuthEveryDayG** — это современный, высокопроизводительный плагин авторизации для серверов **Purpur / Paper 26.2 (Minecraft 1.26.2)**. Написан полностью на **Kotlin** под **Java 25**, что обеспечивает максимальную скорость работы, лаконичность кода и полную совместимость с передовыми технологиями экосистемы Minecraft.

Плагин разработан как легковесная альтернатива AuthMe с упором на современный пользовательский опыт взаимодействия через **Discord 2FA**.

---

## ✨ Ключевые особенности

* 🚀 **Стек нового поколения:** Полностью на Kotlin, скомпилирован под Java 25 для работы на новейших сборках Purpur/Paper 26.2.
* 📦 **Полная автономность (ShadowJar):** Все необходимые библиотеки (включая JDA) вшиты внутрь плагина и изолированы (relocated), что исключает конфликты с другими дискорд-плагинами.
* 📂 **Хранилище SQLite:** Быстрая и надежная база данных `auth.db`, не требующая настройки внешних серверов. Имена пользователей регистронезависимы для предотвращения обхода защиты.
* 🛡️ **Защита неавторизованных сессий:** Полная блокировка перемещений, чата (`AsyncChatEvent`), ломания/постановки блоков и выбрасывания предметов до подтверждения личности.
* 🕒 **Временные сессии:** Настраиваемое время жизни сессии (например, 5 минут). Если игрок перезаходит в пределах этого времени, повторный ввод пароля и 2FA не требуются.
* 🗺️ **Интерактивная 2FA авторизация:** Никаких скучных кодов. Игрок получает в ЛС Discord красивую карточку с метаданными (IP, Время, Страна и Город через GeoIP) и кнопкой **«Авторизовать»**.

## 🛠️ Команды и Права

| Команда | Описание | Доступ |
| :--- | :--- | :--- |
| `/register <пароль> <Discord_ID>` | Регистрация нового аккаунта на сервере | Доступно всем |
| `/login <пароль>` | Авторизация и запуск процесса проверки 2FA | Доступно всем |
| `/2fa help` | Показать меню управления защитой | Доступно всем |
| `/2fa enable` | Включить отправку запросов 2FA при входе | Доступно всем |
| `/2fa disable` | Выключить 2FA (вход только по паролю) | Доступно всем |
| `/2fa unlink` | Полностью отвязать текущий аккаунт Discord | Доступно всем |
| `/2fa <Новый_ID>` | Привязать другой аккаунт Discord | Доступно всем |

*Все команды по умолчанию имеют встроенное право `autheveryday.use` в манифесте `paper-plugin.yml`.*

---

## ⚙️ Конфигурация (`config.yml`)

```yaml
# Токен вашего Discord бота (создается на https://discord.com)
# Обязательно включите "Direct Messages Intent" во вкладке Bot!
discord-token: "YOUR_BOT_TOKEN"

# Время сессии в минутах (если игрок перезашел в течение этого времени, 2FA не запрашивается)
session-duration-minutes: 5

# Включить 2FA принудительно для всех новых игроков сразу при регистрации?
default-2fa-enabled: true
```
